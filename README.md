# Portfolio Web

Este proyecto es mi **portfolio personal**, pensado como un espacio para mostrar proyectos, experiencia y forma de trabajar como developer.

## 🧠 Enfoque del proyecto

Está organizado para que sea fácil de modificar y ampliar. Cada parte tiene una responsabilidad clara y el código intenta mantenerse simple y legible.

Algunas ideas que se han tenido en cuenta:

- Separación clara entre lógica, componentes y configuración  
- Componentes reutilizables y desacoplados  
- Tipado con TypeScript para evitar errores y facilitar refactors  
- Estructura pensada para añadir nuevas secciones o features sin romper lo existente  

## 🎨 Diseño y experiencia

La web es totalmente **responsive**.  
Las animaciones de fondo se han hecho con **Framer Motion**, usando blobs de colores animados.

La UI utiliza **Tailwind CSS** y **shadcn/ui** para mantener consistencia visual y velocidad de desarrollo.

## ✉️ Contacto y emails

El formulario de contacto envía emails usando **Resend** junto con **React Email**.

## 🚀 Tecnologías utilizadas

- **React**
- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion**
- **Resend**
- **React Email**
- **Git & GitHub**
- **Vercel**
- **Visual Studio Code**

## 📦 Deploy

El proyecto está desplegado en **Vercel**, aprovechando el flujo automático de builds y despliegues desde GitHub.

## 📌 Notas finales

Este proyecto sirve tanto como portfolio visual como ejemplo de cómo afronto un proyecto real: código claro, estructura cuidada y facilidad para ampliar sin volverse loca 💜
