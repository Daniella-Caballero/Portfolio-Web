# Portfolio Web

Este proyecto es mi **portfolio personal**, pensado como un espacio para mostrar proyectos, experiencia y forma de trabajar como developer. No es solo una web visual, sino también un proyecto planteado para crecer sin volverse caótico con el tiempo.

## 🧠 Enfoque del proyecto

La aplicación está organizada para que sea fácil de entender, modificar y ampliar. Cada parte tiene una responsabilidad clara y el código intenta mantenerse simple y legible.

Algunas ideas que se han tenido en cuenta:

- Separación clara entre lógica, componentes y configuración  
- Componentes reutilizables y desacoplados  
- Tipado con TypeScript para evitar errores y facilitar refactors  
- Estructura pensada para añadir nuevas secciones o features sin romper lo existente  

Sin sobreingeniería, pero con orden y sentido común.

## 🎨 Diseño y experiencia

La web es **responsive** y está pensada para funcionar bien en desktop y móvil.  
Las animaciones de fondo se han hecho con **Framer Motion**, usando blobs animados de forma suave para dar dinamismo sin distraer.

La UI se apoya en **Tailwind CSS** y **shadcn/ui** para mantener consistencia visual y velocidad de desarrollo.

## ✉️ Contacto y emails

El formulario de contacto envía emails usando **Resend** junto con **React Email**, lo que permite generar emails limpios y mantenibles directamente desde React.

## 🚀 Tecnologías utilizadas

- **React**
- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion**
- **Resend**
- **React Email**

## 📦 Deploy

El proyecto está desplegado en **Vercel**, aprovechando el flujo automático de builds y despliegues desde GitHub.

## 📌 Notas finales

Este proyecto sirve tanto como portfolio visual como ejemplo de cómo afronto un proyecto real: código claro, estructura cuidada y margen para evolucionar sin tener que rehacerlo todo desde cero.
